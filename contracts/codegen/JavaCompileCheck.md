# Java generated-types compile check

The contract Java records are generated by `contracts/codegen/generate.py` into
`contracts/generated/java/`. Full Maven/Gradle packaging (artifact `io.simintero.contracts`)
is deferred to Section C with registry publishing. For Section A we only verify the
generated sources compile.

## Verify

    contracts/codegen/.venv/bin/python contracts/codegen/generate.py
    javac -d /tmp/contracts-java-check $(find contracts/generated/java -name '*.java')

Expected: compiles with no errors (no output).
