# T05 — FHIR API + CapabilityStatement Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Stand up a HAPI FHIR R4 JPA server in `services/interop/` supporting US Core Patient/Practitioner/Coverage/Organization with SMART on FHIR auth, tenant isolation via JWT claim, and a config-generated CapabilityStatement.

**Architecture:** HAPI FHIR JPA Server (Spring Boot) with HAPI request partitioning for tenant isolation; JWT validated by Spring Security OAuth2 resource server against Keycloak JWKS URI; CapabilityStatement generated from `@ConfigurationProperties` (never hand-edited); all interceptors (SmartAuth, Tenant, Audit) registered on the HAPI JPA server bean.

**Tech Stack:** Java 21, Spring Boot 3.3.4, HAPI FHIR 7.4.0, PostgreSQL (HAPI JPA), Spring Security OAuth2 Resource Server, Testcontainers 1.20.1, nimbus-jose-jwt 9.40 (test JWTs).

> **Invariant note:** Every request must include a valid `tenant_id` JWT claim. The `TenantInterceptor` rejects without it with 403. The `SmartAuthInterceptor` rejects unauthenticated requests with 401. Tests must assert both.

```java
package com.simintero.enstellar.interop.config;

import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.security.config.annotation.web.builders.HttpSecurity;
import org.springframework.security.config.annotation.web.configuration.EnableWebSecurity;
import org.springframework.security.config.http.SessionCreationPolicy;
import org.springframework.security.web.SecurityFilterChain;

@Configuration
@EnableWebSecurity
public class SecurityConfig {

    /**
     * Permit /fhir/metadata unauthenticated (SMART discovery).
     * All other /fhir/** endpoints require a valid JWT.
     * Spring Security's oauth2ResourceServer validates the JWT against the configured JWKS URI.
     */
    @Bean
    public SecurityFilterChain securityFilterChain(HttpSecurity http) throws Exception {
        http
            .sessionManagement(sm -> sm.sessionCreationPolicy(SessionCreationPolicy.STATELESS))
            .csrf(csrf -> csrf.disable())
            .authorizeHttpRequests(auth -> auth
                .requestMatchers("/fhir/metadata").permitAll()
                .requestMatchers("/actuator/health").permitAll()
                .requestMatchers("/fhir/**").authenticated()
                .anyRequest().denyAll()
            )
            .oauth2ResourceServer(oauth2 -> oauth2.jwt(jwt -> {}));
        return http.build();
    }
}
```

```java
package com.simintero.enstellar.interop;

import com.nimbusds.jose.jwk.RSAKey;
import org.springframework.boot.test.context.TestConfiguration;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Primary;
import org.springframework.security.oauth2.jwt.JwtDecoder;
import org.springframework.security.oauth2.jwt.NimbusJwtDecoder;

import java.security.KeyPair;
import java.security.KeyPairGenerator;
import java.security.interfaces.RSAPublicKey;
import java.security.interfaces.RSAPrivateKey;

@TestConfiguration
public class TestSecurityConfig {

    /** Shared test RSA key pair. Package-private so FhirTestBase.mintJwt() can sign with the private key. */
    static final RSAKey TEST_RSA_KEY = generateRsaKey();

    @Bean
    @Primary
    public JwtDecoder testJwtDecoder() throws Exception {
        RSAPublicKey publicKey = TEST_RSA_KEY.toRSAPublicKey();
        return NimbusJwtDecoder.withPublicKey(publicKey).build();
    }

    private static RSAKey generateRsaKey() {
        try {
            KeyPairGenerator gen = KeyPairGenerator.getInstance("RSA");
            gen.initialize(2048);
            KeyPair pair = gen.generateKeyPair();
            return new RSAKey.Builder((RSAPublicKey) pair.getPublic())
                    .privateKey((RSAPrivateKey) pair.getPrivate())
                    .keyID("test-key-1")
                    .build();
        } catch (Exception e) {
            throw new RuntimeException("Failed to generate test RSA key", e);
        }
    }
}
```

```java
import com.nimbusds.jwt.JWTClaimsSet;
import com.nimbusds.jwt.SignedJWT;
import org.junit.jupiter.api.BeforeAll;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.boot.test.context.SpringBootTest;
import org.springframework.boot.test.web.client.TestRestTemplate;
import org.springframework.boot.test.web.server.LocalServerPort;
import org.springframework.context.annotation.Import;
import org.springframework.test.context.DynamicPropertyRegistry;
import org.springframework.test.context.DynamicPropertySource;
import org.testcontainers.containers.PostgreSQLContainer;
import org.testcontainers.junit.jupiter.Container;
import org.testcontainers.junit.jupiter.Testcontainers;

import ca.uhn.fhir.jpa.partition.IPartitionLookupSvc;
import ca.uhn.fhir.jpa.entity.PartitionEntity;
import ca.uhn.fhir.rest.api.server.SystemRequestDetails;

import java.util.Date;

/**
 * Base class for all FHIR integration tests.
 *
 * Provides:
 *  - A shared Testcontainers PostgreSQL instance (started once per JVM via static @Container).
 *  - @DynamicPropertySource wiring the datasource to the container.
 *  - TestSecurityConfig import so Spring Security uses the test RSA key instead of Keycloak.
 *  - mintJwt() helper to create signed test JWTs.
 *  - createPartition() helper to register HAPI partitions before tests that create resources.
 */
@SpringBootTest(webEnvironment = SpringBootTest.WebEnvironment.RANDOM_PORT)
@Testcontainers
@Import(TestSecurityConfig.class)
public abstract class FhirTestBase {

    @Container
    static final PostgreSQLContainer<?> POSTGRES = new PostgreSQLContainer<>("postgres:16-alpine")
            .withDatabaseName("hapi_test")
            .withUsername("hapi")
            .withPassword("hapi_secret");

    @DynamicPropertySource
    static void configureDataSource(DynamicPropertyRegistry registry) {
        registry.add("spring.datasource.url", POSTGRES::getJdbcUrl);
        registry.add("spring.datasource.username", POSTGRES::getUsername);
        registry.add("spring.datasource.password", POSTGRES::getPassword);
        // Point JWKS URI at a non-existent host — TestSecurityConfig.testJwtDecoder() takes priority.
        registry.add("spring.security.oauth2.resourceserver.jwt.jwk-set-uri",
                () -> "http://test-issuer-not-used");
    }

    @LocalServerPort
    protected int port;

    @Autowired
    protected TestRestTemplate restTemplate;

    @Autowired
    private IPartitionLookupSvc partitionLookupSvc;

    /**
     * Create a HAPI partition for the given tenant name if it does not already exist.
     * Must be called before creating resources for that tenant.
     */
    protected void createPartition(String tenantName) {
        try {
            partitionLookupSvc.findPartitionByName(tenantName);
        } catch (Exception notFound) {
            PartitionEntity partition = new PartitionEntity();
            partition.setName(tenantName);
            partition.setDescription("Test partition for " + tenantName);
            partitionLookupSvc.createPartition(partition, new SystemRequestDetails());
        }
    }

    /**
     * Create a signed JWT suitable for use in an Authorization: Bearer header.
     *
     * @param tenantId value to set in the "tenant_id" claim; pass null to omit the claim
     * @param scopes   space-separated SMART scopes (e.g. "patient/*.read")
     */
    protected static String mintJwt(String tenantId, String scopes) {
        try {
            JWTClaimsSet.Builder builder = new JWTClaimsSet.Builder()
                    .issuer("http://test-issuer")
                    .subject("test-user")
                    .claim("scope", scopes)
                    .expirationTime(new Date(System.currentTimeMillis() + 3_600_000L));

            if (tenantId != null) {
                builder.claim("tenant_id", tenantId);
            }

            SignedJWT jwt = new SignedJWT(
                    new JWSHeader.Builder(JWSAlgorithm.RS256)
                            .keyID("test-key-1")
                            .build(),
                    builder.build()
            );
            jwt.sign(new RSASSASigner(TestSecurityConfig.TEST_RSA_KEY));
            return jwt.serialize();
        } catch (Exception e) {
            throw new RuntimeException("Failed to mint test JWT", e);
        }
    }

    /** Convenience overload: mints a token with tenant_id="test-tenant" and patient/*.read+write scopes. */
    protected static String mintDefaultJwt() {
        return mintJwt("test-tenant", "patient/*.read patient/*.write");
    }
}
```

## Task 3: TenantInterceptor

**Files:**
- Create: `services/interop/src/main/java/com/simintero/enstellar/interop/interceptors/TenantInterceptor.java`

The `TenantInterceptor` uses HAPI's pointcut system to:
1. Enforce presence of `tenant_id` in the JWT on every FHIR request.
2. Set HAPI's `RequestPartitionId` so that all reads and writes are scoped to that tenant's partition.

- [ ] **Step 3.1: Create `TenantInterceptor.java`**

```java
package com.simintero.enstellar.interop.interceptors;

import ca.uhn.fhir.interceptor.api.Hook;
import ca.uhn.fhir.interceptor.api.Interceptor;
import ca.uhn.fhir.interceptor.api.Pointcut;
import ca.uhn.fhir.interceptor.model.RequestPartitionId;
import ca.uhn.fhir.rest.api.server.RequestDetails;
import ca.uhn.fhir.rest.server.exceptions.ForbiddenOperationException;
import jakarta.servlet.http.HttpServletRequest;
import jakarta.servlet.http.HttpServletResponse;
import org.slf4j.Logger;
import org.slf4j.LoggerFactory;
import org.springframework.security.core.Authentication;
import org.springframework.security.core.context.SecurityContextHolder;
import org.springframework.security.oauth2.jwt.Jwt;
import org.springframework.stereotype.Component;

/**
 * Extracts the {@code tenant_id} claim from the validated JWT and sets HAPI's
 * {@link RequestPartitionId} to isolate all reads and writes to that tenant's partition.
 *
 * If the claim is absent the request is rejected with 403 — this satisfies the
 * invariant that no data operation executes without a tenant context.
 */
@Component
@Interceptor
public class TenantInterceptor {

    private static final Logger LOG = LoggerFactory.getLogger(TenantInterceptor.class);
    static final String TENANT_ID_CLAIM = "tenant_id";

    /**
     * Runs before HAPI routes the request to a resource provider.
     * Validates the tenant_id claim and stores it on the request for downstream use.
     */
    @Hook(Pointcut.SERVER_INCOMING_REQUEST_PRE_HANDLED)
    public void enforceTenantClaim(RequestDetails requestDetails,
                                   HttpServletRequest servletRequest,
                                   HttpServletResponse servletResponse) {
        String tenantId = extractTenantId();
        if (tenantId == null || tenantId.isBlank()) {
            throw new ForbiddenOperationException(
                    "Request rejected: JWT is missing required 'tenant_id' claim.");
        }
        // Store tenant_id for use by providers and audit interceptor.
        requestDetails.setAttribute(TENANT_ID_CLAIM, tenantId);
        LOG.debug("Tenant context set: tenant_id={}", tenantId);
    }

    /**
     * Sets the HAPI partition for CREATE operations (writes are scoped to this partition).
     */
    @Hook(Pointcut.STORAGE_PARTITION_IDENTIFY_CREATE)
    public RequestPartitionId identifyPartitionForCreate(RequestDetails requestDetails) {
        return partitionFromRequest(requestDetails);
    }

    /**
     * Sets the HAPI partition for READ / SEARCH operations (reads only see this partition).
     */
    @Hook(Pointcut.STORAGE_PARTITION_IDENTIFY_READ)
    public RequestPartitionId identifyPartitionForRead(RequestDetails requestDetails) {
        return partitionFromRequest(requestDetails);
    }

    private RequestPartitionId partitionFromRequest(RequestDetails requestDetails) {
        if (requestDetails == null) {
            return RequestPartitionId.defaultPartition();
        }
        Object tenantId = requestDetails.getAttribute(TENANT_ID_CLAIM);
        if (tenantId instanceof String s && !s.isBlank()) {
            return RequestPartitionId.fromPartitionName(s);
        }
        // Partition hooks fire after SERVER_INCOMING_REQUEST_PRE_HANDLED, so tenant_id should
        // already be present. Fallback to default partition for system operations.
        return RequestPartitionId.defaultPartition();
    }

    /** Extract tenant_id from the Spring Security context (JWT already validated by Spring Security). */
    private String extractTenantId() {
        Authentication auth = SecurityContextHolder.getContext().getAuthentication();
        if (auth == null || auth.getPrincipal() == null) {
            return null;
        }
        if (auth.getPrincipal() instanceof Jwt jwt) {
            return jwt.getClaimAsString(TENANT_ID_CLAIM);
        }
        return null;
    }
}
```

```java
/**
 * Bridges Spring Security authentication into HAPI's interceptor chain.
 *
 * Spring Security's filter chain runs before the HAPI servlet and validates the JWT.
 * If the SecurityContext is not authenticated at this point, the filter chain has a bug —
 * but this interceptor provides a defence-in-depth check and produces a proper HAPI
 * OperationOutcome 401 response rather than a generic Spring error page.
 *
 * /fhir/metadata is excluded by SecurityConfig.permitAll() so this interceptor is
 * not reached for that path.

@Component
@Interceptor
public class SmartAuthInterceptor {

                                      HttpServletRequest servletRequest,
                                      HttpServletResponse servletResponse) {
        Authentication auth = SecurityContextHolder.getContext().getAuthentication();
        if (auth == null || !auth.isAuthenticated() || "anonymousUser".equals(auth.getPrincipal())) {
            throw new AuthenticationException(
                    "No valid SMART on FHIR token. Provide a Bearer token in the Authorization header.");
        }
    }
```

```java
import ca.uhn.fhir.interceptor.api.Hook;
import ca.uhn.fhir.interceptor.api.Interceptor;

import org.springframework.security.oauth2.jwt.Jwt;

/**
 * Writes a structured JSON audit event to stdout on each FHIR response.
 *
 * Output format (one JSON object per line, no PHI):
 * {"ts":"...","tenant_id":"...","subject":"...","method":"...","path":"...","status":200}
 *
 * PHI rule: do not log resource content, patient IDs, or member data.
 * The audit record contains only tenant_id, subject (user), method, path, and HTTP status.
 */
@Component
@Interceptor
public class AuditInterceptor {

    private static final Logger AUDIT = LoggerFactory.getLogger("AUDIT");

    @Hook(Pointcut.SERVER_OUTGOING_RESPONSE)
    public void logAuditEvent(RequestDetails requestDetails,
                              HttpServletRequest servletRequest,
                              HttpServletResponse servletResponse) {
        if (requestDetails == null) return;

        String tenantId = "(none)";
        String subject = "(anonymous)";

        Object tenantAttr = requestDetails.getAttribute(TenantInterceptor.TENANT_ID_CLAIM);
        if (tenantAttr instanceof String s) {
            tenantId = s;
        }

        Authentication auth = SecurityContextHolder.getContext().getAuthentication();
        if (auth != null && auth.getPrincipal() instanceof Jwt jwt) {
            subject = jwt.getSubject() != null ? jwt.getSubject() : "(no-sub)";
        }

        int status = servletResponse != null ? servletResponse.getStatus() : -1;
        String method = servletRequest != null ? servletRequest.getMethod() : "?";
        String path = requestDetails.getRequestPath();

        // Structured JSON audit line — keep PHI-free
        AUDIT.info("{{\"ts\":\"{}\",\"tenant_id\":\"{}\",\"subject\":\"{}\",\"method\":\"{}\",\"path\":\"{}\",\"status\":{}}}",
                Instant.now(), tenantId, subject, method, path, status);
    }
}
```

```java
package com.simintero.enstellar.interop.conformance;

import ca.uhn.fhir.rest.server.RestfulServer;
import ca.uhn.fhir.rest.server.provider.ServerCapabilityStatementProvider;
import com.simintero.enstellar.interop.config.FhirCapabilityProperties;
import jakarta.servlet.http.HttpServletRequest;
import org.hl7.fhir.r4.model.CapabilityStatement;
import org.hl7.fhir.r4.model.Enumerations.PublicationStatus;

import java.util.Date;

/**
 * Generates the FHIR CapabilityStatement from {@link FhirCapabilityProperties}.
 *
 * This class is the ONLY place where the CapabilityStatement is written.
 * The invariant "never hand-edit the CapabilityStatement" is enforced by
 * having all content come from application.yml via FhirCapabilityProperties.
 *
 * The parent {@link ServerCapabilityStatementProvider} is NOT called; we build
 * the statement from scratch so that only explicitly configured resources appear.
 *
 * NOT a Spring @Component — instantiated manually in HapiServerConfig.fhirServer()
 * after the RestfulServer is constructed, so the server reference is available.
 */
public class EnstellarCapabilityStatementProvider extends ServerCapabilityStatementProvider {

    private final FhirCapabilityProperties props;

    public EnstellarCapabilityStatementProvider(RestfulServer server,
                                                FhirCapabilityProperties props) {
        super(server);
        this.props = props;
    }

    @Override
    public CapabilityStatement getServerConformance(HttpServletRequest request,
                                                    ca.uhn.fhir.rest.api.server.RequestDetails details) {
        CapabilityStatement cs = new CapabilityStatement();
        cs.setStatus(PublicationStatus.ACTIVE);
        cs.setDate(new Date());
        cs.setPublisher(props.getPublisher());
        cs.setKind(CapabilityStatement.CapabilityStatementKind.INSTANCE);
        cs.setFhirVersion(org.hl7.fhir.r4.model.Enumerations.FHIRVersion._4_0_1);
        cs.addFormat("application/fhir+json");
        cs.addFormat("application/fhir+xml");

        CapabilityStatement.CapabilityStatementRestComponent rest =
                cs.addRest().setMode(CapabilityStatement.RestfulCapabilityMode.SERVER);

        for (FhirCapabilityProperties.ResourceConfig rc : props.getResources()) {
            CapabilityStatement.CapabilityStatementRestResourceComponent resource = rest.addResource();
            resource.setType(rc.getType());

            if (rc.getProfile() != null && !rc.getProfile().isBlank()) {
                resource.setProfile(rc.getProfile());
            }

            for (String interaction : rc.getInteractions()) {
                CapabilityStatement.ResourceInteractionComponent interactionComp =
                        resource.addInteraction();
                interactionComp.setCode(
                    CapabilityStatement.TypeRestfulInteraction.fromCode(interaction)
                );
            }
        }

        return cs;
    }
}
```

```java
package com.simintero.enstellar.interop.config;

import ca.uhn.fhir.context.FhirContext;
import ca.uhn.fhir.rest.api.EncodingEnum;
import ca.uhn.fhir.rest.server.RestfulServer;
import com.simintero.enstellar.interop.conformance.EnstellarCapabilityStatementProvider;
import com.simintero.enstellar.interop.interceptors.AuditInterceptor;
import com.simintero.enstellar.interop.interceptors.SmartAuthInterceptor;
import com.simintero.enstellar.interop.interceptors.TenantInterceptor;
import com.simintero.enstellar.interop.providers.*;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.boot.web.servlet.ServletRegistrationBean;
import org.springframework.context.ApplicationContext;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.context.annotation.Lazy;

import java.util.List;

/**
 * Configures the HAPI FHIR server bean.
 *
 * Interceptor registration order matters:
 *   1. SmartAuthInterceptor — reject unauthenticated requests first (401)
 *   2. TenantInterceptor — extract tenant context and set partition (403 if missing)
 *   3. AuditInterceptor — log all responses (fires after everything else)
 *
 * The CapabilityStatement is generated from FhirCapabilityProperties; never from runtime
 * introspection of registered providers.
 */
@Configuration
public class HapiServerConfig {

    @Autowired
    private ApplicationContext applicationContext;

    @Autowired
    private SmartAuthInterceptor smartAuthInterceptor;

    @Autowired
    private TenantInterceptor tenantInterceptor;

    @Autowired
    private AuditInterceptor auditInterceptor;

    @Autowired
    private FhirCapabilityProperties capabilityProperties;

    @Autowired
    private PatientResourceProvider patientResourceProvider;

    @Autowired
    private PractitionerResourceProvider practitionerResourceProvider;

    @Autowired
    private CoverageResourceProvider coverageResourceProvider;

    @Autowired
    private OrganizationResourceProvider organizationResourceProvider;

    @Bean
    public RestfulServer fhirServer(FhirContext fhirContext) {
        RestfulServer server = new RestfulServer(fhirContext);
        server.setDefaultResponseEncoding(EncodingEnum.JSON);
        server.setDefaultPrettyPrint(false);

        // Interceptors in evaluation order (SmartAuth → Tenant → Audit).
        server.registerInterceptor(smartAuthInterceptor);
        server.registerInterceptor(tenantInterceptor);
        server.registerInterceptor(auditInterceptor);

        // Resource providers.
        server.registerProviders(List.of(
                patientResourceProvider,
                practitionerResourceProvider,
                coverageResourceProvider,
                organizationResourceProvider
        ));

        // CapabilityStatement provider — generated from config, never hand-edited.
        EnstellarCapabilityStatementProvider csProvider =
                new EnstellarCapabilityStatementProvider(server, capabilityProperties);
        server.setServerConformanceProvider(csProvider);

        return server;
    }

    @Bean
    public ServletRegistrationBean<RestfulServer> fhirServletRegistration(RestfulServer server) {
        ServletRegistrationBean<RestfulServer> reg = new ServletRegistrationBean<>(server, "/fhir/*");
        reg.setLoadOnStartup(1);
        return reg;
    }
}
```

**Note on HAPI JPA DAOs:** The resource providers `@Autowired IFhirResourceDao<Patient>` etc. are provided by HAPI FHIR's JPA auto-configuration. When `hapi-fhir-jpaserver-base` is on the classpath alongside `hapi-fhir-spring-boot-starter`, HAPI's `FhirAutoConfiguration` registers `IFhirResourceDao` beans for all R4 resource types. If the context fails to find these beans, add the following to `HapiServerConfig` or a separate `@Configuration`:

```java
// @Import(ca.uhn.fhir.jpa.config.r4.JpaR4Config.class)
```

```java
package com.simintero.enstellar.interop;

import com.fasterxml.jackson.databind.JsonNode;
import com.fasterxml.jackson.databind.ObjectMapper;
import org.junit.jupiter.api.BeforeEach;
import org.junit.jupiter.api.Test;
import org.springframework.http.*;

import static org.assertj.core.api.Assertions.assertThat;

class PatientResourceIT extends FhirTestBase {

    private static final ObjectMapper JSON = new ObjectMapper();
    private static final String TENANT = "patient-it-tenant";

    @BeforeEach
    void ensurePartition() {
        createPartition(TENANT);
    }

    @Test
    void create_and_read_patient_by_id() throws Exception {
        String token = mintJwt(TENANT, "patient/*.read patient/*.write");
        HttpHeaders headers = authHeaders(token);

        // Build a minimal US Core–compliant Patient resource.
        String patientJson = """
                {
                  "resourceType": "Patient",
                  "meta": {
                    "profile": ["http://hl7.org/fhir/us/core/StructureDefinition/us-core-patient"]
                  },
                  "identifier": [{"system": "http://example.org/mrn", "value": "MRN-001"}],
                  "name": [{"family": "Smith", "given": ["Alice"]}],
                  "gender": "female",
                  "birthDate": "1985-03-22"
                }
                """;

        // Create
        ResponseEntity<String> createResponse = restTemplate.exchange(
                "http://localhost:" + port + "/fhir/Patient",
                HttpMethod.POST,
                new HttpEntity<>(patientJson, headers),
                String.class
        );
        assertThat(createResponse.getStatusCode()).isEqualTo(HttpStatus.CREATED);
        String locationHeader = createResponse.getHeaders().getFirst("Location");
        assertThat(locationHeader).isNotNull();

        // Extract the ID from the Location header (e.g., /fhir/Patient/123/_history/1)
        String patientId = extractIdFromLocation(locationHeader);

        // Read back
        ResponseEntity<String> readResponse = restTemplate.exchange(
                "http://localhost:" + port + "/fhir/Patient/" + patientId,
                HttpMethod.GET,
                new HttpEntity<>(headers),
                String.class
        );
        assertThat(readResponse.getStatusCode()).isEqualTo(HttpStatus.OK);

        JsonNode patient = JSON.readTree(readResponse.getBody());
        assertThat(patient.path("resourceType").asText()).isEqualTo("Patient");
        assertThat(patient.path("name").path(0).path("family").asText()).isEqualTo("Smith");
    }

    @Test
    void search_patient_by_family_name_returns_matching_only() throws Exception {
        String token = mintJwt(TENANT, "patient/*.read patient/*.write");
        HttpHeaders headers = authHeaders(token);

        // Create two patients with distinct family names.
        createPatient(headers, "Jones", "Bob");
        createPatient(headers, "Williams", "Carol");

        // Search by family=Jones
        ResponseEntity<String> searchResponse = restTemplate.exchange(
                "http://localhost:" + port + "/fhir/Patient?family=Jones",
                HttpMethod.GET,
                new HttpEntity<>(headers),
                String.class
        );
        assertThat(searchResponse.getStatusCode()).isEqualTo(HttpStatus.OK);

        JsonNode bundle = JSON.readTree(searchResponse.getBody());
        assertThat(bundle.path("resourceType").asText()).isEqualTo("Bundle");

        // All returned entries must have family=Jones.
        for (JsonNode entry : bundle.path("entry")) {
            String family = entry.path("resource").path("name").path(0).path("family").asText();
            assertThat(family).isEqualTo("Jones");
        }
    }

    @Test
    void search_patient_by_identifier_returns_correct_patient() throws Exception {
        String token = mintJwt(TENANT, "patient/*.read patient/*.write");
        HttpHeaders headers = authHeaders(token);

        createPatient(headers, "Taylor", "David");

        ResponseEntity<String> searchResponse = restTemplate.exchange(
                "http://localhost:" + port + "/fhir/Patient?identifier=http://example.org/mrn|MRN-TAYLOR",
                HttpMethod.GET,
                new HttpEntity<>(headers),
                String.class
        );
        assertThat(searchResponse.getStatusCode()).isEqualTo(HttpStatus.OK);

        JsonNode bundle = JSON.readTree(searchResponse.getBody());
        // At least one entry with this identifier.
        boolean found = false;
        for (JsonNode entry : bundle.path("entry")) {
            for (JsonNode id : entry.path("resource").path("identifier")) {
                if ("MRN-TAYLOR".equals(id.path("value").asText())) {
                    found = true;
                }
            }
        }
        assertThat(found).as("Patient with identifier MRN-TAYLOR should appear in search results").isTrue();
    }

    // ─── helpers ────────────────────────────────────────────────────────────

    private HttpHeaders authHeaders(String token) {
        HttpHeaders h = new HttpHeaders();
        h.setBearerAuth(token);
        h.setContentType(MediaType.valueOf("application/fhir+json"));
        h.setAccept(java.util.List.of(MediaType.valueOf("application/fhir+json")));
        return h;
    }

    private void createPatient(HttpHeaders headers, String family, String given) {
        String mrn = "MRN-" + family.toUpperCase();
        String json = """
                {
                  "resourceType": "Patient",
                  "identifier": [{"system": "http://example.org/mrn", "value": "%s"}],
                  "name": [{"family": "%s", "given": ["%s"]}],
                  "gender": "unknown",
                  "birthDate": "2000-01-01"
                }
                """.formatted(mrn, family, given);
        restTemplate.exchange(
                "http://localhost:" + port + "/fhir/Patient",
                HttpMethod.POST,
                new HttpEntity<>(json, headers),
                String.class
        );
    }

    private String extractIdFromLocation(String location) {
        // Location: http://localhost:<port>/fhir/Patient/<id>/_history/1
        String[] segments = location.split("/");
        // Find "Patient" segment and take the next one.
        for (int i = 0; i < segments.length - 1; i++) {
            if ("Patient".equals(segments[i])) {
                return segments[i + 1];
            }
        }
        throw new IllegalArgumentException("Cannot extract ID from Location header: " + location);
    }
}
```

## Troubleshooting Guide

**`IFhirResourceDao<Patient>` cannot be found / NoSuchBeanDefinitionException:**
HAPI's JPA DAO beans require HAPI's internal JPA configuration to be loaded. If they are not auto-configured by `hapi-fhir-spring-boot-starter`, add to `HapiServerConfig.java`:
```java
@Import(ca.uhn.fhir.jpa.config.r4.JpaR4Config.class)
```
And add to `InteropApplication.java`:
```java
@EntityScan(basePackages = {"ca.uhn.fhir.jpa.entity", "ca.uhn.fhir.jpa.model.entity"})
@EnableJpaRepositories(basePackages = {"ca.uhn.fhir.jpa.dao", "ca.uhn.fhir.jpa.model"})
```

**Hibernate schema creation fails with `HapiFhirPostgres94Dialect` not found:**
Verify the `hapi-fhir-jpaserver-base:7.4.0` artifact is on the classpath. Run:
```bash
cd services/interop && ./gradlew dependencies --configuration runtimeClasspath | grep hapi-fhir-jpaserver
```
Expected: `ca.uhn.hapi.fhir:hapi-fhir-jpaserver-base:7.4.0`

**Testcontainers: Docker not available in CI:**
GitHub Actions ubuntu runners include Docker. For local dev, ensure Docker Desktop or Docker Engine is running before `./gradlew test`.

**`RequestPartitionId.fromPartitionName()` throws PartitionNotFoundException:**
The partition must be registered in HAPI's partition registry before the first resource is written to it. Ensure `createPartition()` is called in `@BeforeEach` for every test that creates resources. For non-test flows, the `TenantInterceptor` triggers partition-identify hooks — if the partition does not exist, add auto-creation logic to the interceptor's `identifyPartitionForCreate` method using `IPartitionLookupSvc`.

**`ServerCapabilityStatementProvider` constructor requires `RestfulServer` but server is not yet initialized:**
The `EnstellarCapabilityStatementProvider` is constructed inside `HapiServerConfig.fhirServer()` after the server is created, so the server reference is valid at that point. Do not inject it as a Spring bean directly — construct it manually inside `fhirServer()`.

**Spring Security tries to fetch JWKS URI during test startup:**
Verify `TestSecurityConfig` is imported (`@Import(TestSecurityConfig.class)`) in `FhirTestBase` and that `@Primary` is on the `testJwtDecoder()` bean. The `@DynamicPropertySource` sets `spring.security.oauth2.resourceserver.jwt.jwk-set-uri` to a dummy value as a secondary guard.
